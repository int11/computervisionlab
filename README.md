# Computer Vision Lab Website

Sungkyunkwan University (SKKU) Computer Vision Lab official website.

## Overview
This website showcases the research projects, members, and publications of the Computer Vision Lab at SKKU.

## Technologies Used
- Next.js 15
- Tailwind CSS
- TypeScript
- GitHub Pages

## Local Development
Run the following commands to start the development server:
```bash
npm install
npm run dev
```

## Deployment
Automatic deployment is configured via GitHub Actions. Any push to the `main` branch will trigger a build and deploy.
